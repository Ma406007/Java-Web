什么是HTML?
HTML(HyperText Markup Language):超文本标记语言,一种标记语言,包括一系列标签,通过这些标签可以将网络上的文档格式统一从而使分散的Internet资源连接为一个逻辑整体
  超文本:超越了文本的限制,比普通文本更强大,除了文字信息还可以定义图片,音频,视频等内容
  标记语言:由标签"<标签名>"构成的语言
  细节:
    1.HTML标签都说预定义好的.例如:使用<h1>展示标题,使用<img>展示图片,使用<video>展示视频
    2.HTML代码可以直接在浏览器中运行,HTML标签由浏览器解析

HTML的主体框架:
<html>
  <head>
    网页头部:用来存放给浏览器看的信息,如:CSS样式
  </head>
  <body>
    网页主体:用来存放给用户看的信息,如:文字,图片,视频
  </body>
</html>


什么是CSS?
CSS(Cascading Style Sheet):层叠样式表,一种用来表现HTML或XML等文件样式的计算机语言.CSS不仅可以静态地修饰网页,还可以配合各种脚本语言动态地对网页各元素进行格式化
